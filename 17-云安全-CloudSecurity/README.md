# ☁️ 阶段十七：云安全 (Cloud Security)

## 目标
系统化掌握主流云平台（AWS、Azure、GCP）的安全评估方法，包括云基础设施配置审计、IAM权限分析、云存储安全、网络隔离检查和合规性评估。

## 核心技能清单
| # | 技能名称 | 难度 | 推荐工具 |
|:---:|:---|:---:|:---|
| 1 | ☁️ AWS安全评估 | ★★★★ | Prowler, ScoutSuite, AWS Config |
| 2 | 🌤️ Azure安全评估 | ★★★★ | Azure Security Center, Azucar, ScoutSuite |
| 3 | 🔵 GCP安全评估 | ★★★★ | Forseti Security, GCP Inspector, ScoutSuite |
| 4 | 🔑 云IAM权限与访问控制审计 | ★★★ | AWS IAM Access Analyzer, Azure AD |
| 5 | 💾 云存储安全配置审计 | ★★★ | S3 Scanner, CloudSploit, GCloud CLI |
| 6 | 🌐 云网络与WAF安全 | ★★★★ | CloudSploit, Security Groups Check |
| 7 | ⚡ 无服务器架构安全 | ★★★★ | AWS Lambda Check, Serverless Framework |
| 8 | 🔄 多云安全策略评估 | ★★★★★ | Prisma Cloud, Wiz, Orca Security |
