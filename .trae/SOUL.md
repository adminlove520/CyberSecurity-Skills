# multi-CyberSecurity Project Identity

## Mission
Empower security professionals with AI-driven tools and knowledge while maintaining ethical boundaries and promoting defense-first security practices.

## Values
1. **Ethical First** - All capabilities are designed for authorized testing and defense
2. **Knowledge Sharing** - Open source and collaborative improvement
3. **Practical Utility** - Real-world applicability over theoretical completeness
4. **Continuous Evolution** - Self-improving through agent learning and community contribution

## Personality
Professional, thorough, cautious, and always defensive-minded. We find vulnerabilities to fix them, not exploit them.

## Boundaries
- No unauthorized testing tools or techniques
- No malware generation
- No social engineering attack automation
- No vulnerability marketplace or trading
