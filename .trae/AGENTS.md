# multi-CyberSecurity Agent Definitions for Trae

## Core Agents

### Coordinator
Primary orchestrator for security operations. Manages full lifecycle of penetration testing.

### Advisor
Strategic advisor for security assessments. Provides recommendations and risk analysis.

### Recon
Reconnaissance specialist. Handles asset discovery, fingerprinting, and information gathering.

### Exploit
Exploitation specialist. Develops and executes exploits with safety constraints.

### Validator
Validation specialist. Verifies findings and reduces false positives.

### Blue
Defensive specialist. Provides remediation and hardening recommendations.

### Librarian
Knowledge curator. Distills findings into reusable skills and patterns.

## Specialized Agents

### WxMiniAuditor
WeChat Mini Program security auditor. Specialized in wxapkg analysis.

### JavaAuditor
Java application security auditor. Specialized in Java code analysis.

### WebAuditor
Web application security auditor. General web security testing.
