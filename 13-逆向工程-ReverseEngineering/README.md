# ⚙️ 阶段十三：逆向工程 (Reverse Engineering)

## 目标
对二进制程序、恶意软件进行逆向分析，理解其工作原理、发现漏洞或提取关键逻辑。

## 核心技能清单
| # | 技能名称 | 难度 | 推荐工具 |
|:---:|:---|:---:|:---|
| 1 | 静态逆向分析 | ★★★★ | IDA Pro, Ghidra, Radare2, Binary Ninja |
| 2 | 动态调试分析 | ★★★★ | x64dbg, OllyDbg, GDB, WinDbg |
| 3 | 恶意软件分析 | ★★★★★ | Cuckoo Sandbox, CAPE, 动态调试 |
