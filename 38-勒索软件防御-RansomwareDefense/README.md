# 勒索软件防御 (Ransomware Defense)

## 概述

勒索软件是最具破坏力的网络威胁之一，防御需要覆盖攻击链检测、应急响应和主动加固。本模块覆盖勒索软件攻击链分析、应急响应与恢复流程、以及反勒索软件加固与备份策略。
