# 🔐 阶段二十二：数据安全与隐私保护 (Data Security & Privacy)

## 目标
针对组织的数据安全管理体系进行全面审查，涵盖数据分类分级、数据防泄漏（DLP）、数据脱敏与匿名化、隐私合规评估（GDPR/个保法）及数据库安全。

## 核心技能清单
| # | 技能名称 | 难度 | 推荐工具 |
|:---:|:---|:---:|:---|
| 1 | 📋 数据分类与分级保护 | ★★★ | Data Classification Toolkit, Varonis |
| 2 | 🛡️ DLP数据防泄漏策略 | ★★★★ | Microsoft DLP, Symantec DLP, Digital Guardian |
| 3 | 🔏 数据脱敏与匿名化 | ★★★★ | ARX Data Anonymizer, Privacy Analytics |
| 4 | 🌍 GDPR/个保法合规评估 | ★★★★★ | OneTrust, TrustArc, DPIA Toolkit |
| 5 | 🗄️ 数据库安全与加密 | ★★★ | Vault, CipherTrust, SQL Audit Tools |
| 6 | 📝 隐私影响评估PIA | ★★★★ | PIA Template, CNIL PIA Tool |
