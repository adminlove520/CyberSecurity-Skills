# 🏭 阶段十九：工控安全 (ICS/OT Security)

## 目标
针对工业控制系统和运营技术环境进行安全评估与防护，涵盖SCADA、DCS、PLC/RTU等工控设备的安全测试，遵循IEC 62443等国际标准。

## 核心技能清单
| # | 技能名称 | 难度 | 推荐工具 |
|:---:|:---|:---:|:---|
| 1 | ⚙️ SCADA系统安全评估 | ★★★★ | PLCScan, Modbus/TCP Scanner, Shodan |
| 2 | 🔧 PLC与RTU安全测试 | ★★★★★ | MSPT, ISF (ICS Exploitation Framework) |
| 3 | 🌐 工控网络协议安全 | ★★★★ | Wireshark, Scapy, Zabbix |
| 4 | 📋 工控安全合规审计（IEC 62443） | ★★★★ | IEC 62443 Checklist, Compliance Scanner |
| 5 | 🚨 工控安全应急预案 | ★★★★★ | ICS-CERT Tools, Dragos Platform |
| 6 | 🔒 工业防火墙与网络分段 | ★★★★ | Tofino Security, Claroty, Nozomi |
