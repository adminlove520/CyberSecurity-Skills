# 🤖 智能体团队 (Agent Team)

## 角色详解

### 1. Coordinator (任务编排者)
- **职责**: 任务拆解、委派与进度跟踪。
- **输入**: 用户原始需求（如 "对 target.com 进行渗透测试"）。
- **输出**: 结构化的子任务列表于 `MISSION_CONTROL.md`。

### 2. Advisor (战略顾问)
- **职责**: 策略调整、WAF 绕过建议、失败分析。
- **触发条件**: 当执行 Agent 连续 3 次尝试同一漏洞失败时自动介入。
- **核心能力**: 自我反思 (Reflexion) 与策略漂移。

### 3. Recon (侦察专家)
- **职责**: 主被动资产发现、指纹识别、攻击面测绘。
- **集成工具**: Subfinder, Nmap, Gobuster, Amass。

### 4. Exploit (漏洞利用专家)
- **职责**: 漏洞验证与利用、权限提升。
- **核心逻辑**: 基于 `03-漏洞利用` 模块匹配最合适的 PoC。

### 5. Validator (验证专家)
- **职责**: 去误报、证据提取、PoC 录屏/截图。
- **集成工具**: Burp Suite (REST API), sqlmap。

### 6. Blue (蓝队/防护专家)
- **职责**: 修复方案生成、系统加固建议、应急响应。

### 7. Librarian (技能沉淀者)
- **职责**: 任务后的知识蒸馏、技能库自动更新。
