# 威胁情报 (Threat Intelligence)

## 概述

威胁情报通过收集、分析和共享攻击者信息，帮助组织主动防御。本模块覆盖情报馈入管理、MISP 平台部署、APT 组织分析和情报驱动的安全运营决策。

## 关键工具

- MISP — 威胁情报共享平台
- TAXII/STIX — 情报交换协议
- MITRE ATT&CK — 攻击知识库
- VirusTotal / AlienVault OTX — 威胁数据源
- Mermaid Threat — 攻击流可视化
