# 🧹 阶段八：痕迹清除 (Covering Tracks)

## 目标
清除渗透过程中的操作痕迹，防止被溯源检测和取证分析。

## 核心技能清单
| # | 技能名称 | 难度 | 推荐工具 |
|:---:|:---|:---:|:---|
| 1 | 痕迹清除与反取证 | ★★★ | 日志清除, 文件隐藏, 时间戳修改, 反取证技术 |
| 2 | 进程注入与代码注入 | ★★★★ | Cobalt Strike, Meterpreter, PowerSploit |
| 3 | 代码混淆与反分析 | ★★★★ | ConfuserEx, Obfuscator-LLVM, UPX, VMP |
| 4 | AMSI绕过与EDR规避 | ★★★★★ | SysWhispers, AMSI.fail, SharpSploit |
