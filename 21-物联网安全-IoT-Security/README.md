# 📡 阶段二十一：物联网安全 (IoT Security)

## 目标
覆盖物联网设备生命周期的安全评估方法，包括固件逆向分析、无线协议测试（BLE/Zigbee/Z-Wave）、嵌入式硬件安全、IoT通信安全和云平台安全审计。

## 核心技能清单
| # | 技能名称 | 难度 | 推荐工具 |
|:---:|:---|:---:|:---|
| 1 | 🔧 固件逆向与分析 | ★★★★★ | Binwalk, Firmadyne, Ghidra, QEMU |
| 2 | 📶 BLE/Zigbee/Z-Wave安全测试 | ★★★★ | BLEAH, GATTool, Z3sec, HackRF |
| 3 | 🌐 物联网通信协议安全 | ★★★★ | MQTT Pwn, MQTTSA, Wireshark |
| 4 | ⚙️ 嵌入式设备硬件安全测试 | ★★★★★ | JTAGulator, OpenOCD, Saleae Logic |
| 5 | ☁️ 物联网平台与云安全 | ★★★★ | AWS IoT Auditor, Azurite |
| 6 | 🏠 智能家居与车联网安全 | ★★★★ | Home Assistant Audit, CANtact |
