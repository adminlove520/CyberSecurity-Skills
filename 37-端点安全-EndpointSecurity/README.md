# 端点安全 (Endpoint Security)

## 概述

端点是攻击者最常利用的入口，也是安全防御的第一道防线。本模块覆盖 EDR 部署与检测规则、无文件恶意软件与 LOLBins 检测、端点加固与合规基线、以及移动设备安全与 MDM。

- CrowdStrike Falcon / SentinelOne — EDR
