# 安全治理与合规 (Governance & Compliance)

## 概述

安全治理是建立安全战略、策略和度量的顶层框架，合规确保符合法规和行业标准。本模块覆盖安全框架与合规审计、风险管理与安全度量、以及安全策略体系与安全意识培训。
