# 数字取证 (Digital Forensics)

## 概述

数字取证是调查安全事件、收集电子证据、支持法律诉讼的关键能力。本模块覆盖磁盘镜像、内存分析、Windows/Linux 取证分析和浏览器邮件取证技术。
