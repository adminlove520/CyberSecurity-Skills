# 🔄 阶段十八：DevSecOps与安全开发生命周期 (DevSecOps/SDLC)

## 目标
将安全性内嵌到软件开发生命周期的每个阶段，实现"左移安全"实践，包括CI/CD管道安全审计、基础设施即代码安全扫描、SAST/DAST集成和软件供应链安全保障。

## 核心技能清单
| # | 技能名称 | 难度 | 推荐工具 |
|:---:|:---|:---:|:---|
| 1 | 🔗 CI/CD管道安全审计 | ★★★★ | GitLab CI/CD Security, Jenkins Security |
| 2 | 📦 IaC安全扫描 | ★★★ | Checkov, Terrascan, tfsec |
| 3 | 🔍 SAST静态应用安全测试 | ★★★★ | SonarQube, Semgrep, CodeQL |
| 4 | 🌐 DAST动态应用安全测试 | ★★★★ | OWASP ZAP, Burp Suite, Acunetix |
| 5 | 🔗 软件供应链安全 | ★★★★ | Snyk, Dependabot, Trivy, SBOM |
| 6 | 🧠 安全需求与威胁建模 | ★★★★ | STRIDE, PASTA, Threat Dragon |
