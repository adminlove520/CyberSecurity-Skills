# 密码学与PKI (Cryptography & PKI)

## 概述

密码学是网络安全的数学基础，PKI 是信任体系的基石。本模块覆盖 TLS/SSL 安全配置、PKI 架构与证书管理、以及加密算法与密钥管理。

## 关键工具

- OpenSSL — 加密工具集
- certbot / acme.sh — 证书自动化
- CFSSL — 证书签发工具
- HashiCorp Vault — 密钥管理
- HSM (YubiHSM / CloudHSM) — 硬件安全模块
