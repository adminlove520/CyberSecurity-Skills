# 身份与访问管理 (Identity & Access Management)

## 概述

身份与访问管理（IAM）是安全架构的核心支柱。本模块覆盖企业 IAM 策略架构、特权账号管理（PAM）、云 IAM 与联邦认证、以及 AD 域安全与攻击路径分析。

## 关键工具

- Azure AD / Okta — 身份平台
- HashiCorp Vault — 特权账号管理
- AWS IAM / GCP IAM — 云 IAM
- BloodHound / PingCastle — AD 攻击路径分析
- Terraform — IAM 即代码
