# OpenClaw Platform Configuration

## Files
- `AGENTS.md` - OpenClaw-specific agent definitions
- `IDENTITY.md` - Agent identity for OpenClaw
- `SOUL.md` - Project philosophy and values
- `RULES.md` - Coding and security rules

## OpenClaw Integration
This configuration enables CyberSecurity-Skills to work with OpenClaw IDE.

## Usage
Copy the contents to your OpenClaw project configuration or reference via:
```
openclaw config import --source CyberSecurity-Skills/.openclaw
```
