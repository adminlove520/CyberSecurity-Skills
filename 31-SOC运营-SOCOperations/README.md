# SOC运营 (SOC Operations)

## 概述

安全运营中心（SOC）是组织安全防御的前线。本模块覆盖 SIEM 告警规则与关联分析、事件分级与响应流程、安全自动化与编排（SOAR）以及 SOC 运营效能度量。

## 关键工具

- Splunk / Elastic SIEM — 日志管理与分析
- Phantom / Splunk SOAR — 安全编排
- Jira / ServiceNow — 事件跟踪
- Slack / Teams — 告警通知
- Grafana — 运营仪表盘
