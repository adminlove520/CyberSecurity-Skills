# API安全 (API Security)

## 概述

API 是现代应用通信的 backbone，也成为攻击者的主要目标。本模块覆盖 OWASP API 安全测试、API 认证与授权安全、以及 GraphQL 与微服务 API 安全防护。

## 技能列表

| # | 技能名称 | 难度 |
|:---:|---------|:---:|
| 1 | [OWASP API安全测试](skills/OWASP API安全测试-OWASPAPISecurityTesting.md) | ★★★★ |
| 2 | [API认证与授权安全](skills/API认证与授权安全-APIAuthAuthorizationSecurity.md) | ★★★★ |
| 3 | [GraphQL与微服务API安全](skills/GraphQL与微服务API安全-GraphQLMicroserviceAPISecurity.md) | ★★★★ |

## 关键工具

- Burp Suite / Postman — API 测试
- OWASP ZAP — 自动化扫描
- jwt.io / jose — JWT 工具
- Istio / Envoy — 服务网格安全
- Authentik / Keycloak — 身份代理
