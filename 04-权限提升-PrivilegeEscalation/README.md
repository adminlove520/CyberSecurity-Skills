# ⬆️ 阶段四：权限提升 (Privilege Escalation)

## 目标
从低权限用户提升至管理员/root权限，获取最高系统控制权。

| 3 | 内核漏洞与服务配置错误提权 | ★★★★ | 漏洞利用代码, MSF模块 |
