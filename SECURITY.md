# 安全策略 (Security Policy)

## 🔐 漏洞报告

如果您在使用本项目时发现安全相关的问题（如：
- 内容中包含敏感信息泄露
- 推荐的工具/命令存在安全隐患
- 链接指向恶意或过期资源

请通过以下方式负责任地报告：

### 报告流程

1. **不要公开发布** — 请勿在公开 Issue 中披露安全漏洞
2. **发送报告** — 通过 Atomgit Issues 创建私有 Issue（如支持）或发送至项目维护者
3. **预期响应** — 我们会在 72 小时内确认收到报告

### 报告内容

请包含以下信息：
- 问题描述
- 受影响文件及具体位置
- 严重程度评估
- 修复建议（如有）

## ✅ 安全最佳实践

本项目内容仅供**授权的安全测试与合法的安全教育**使用。

### 使用者须知

- ⚠️ 始终在获得明确授权后再进行安全测试
- ⚠️ 请遵守当地法律法规
- ⚠️ 所有工具和技术仅用于教育目的

### 维护者承诺

- 所有内容经过技术审核
- 不包含真实IP、密码、密钥等敏感信息
- 定期审查和更新参考链接
